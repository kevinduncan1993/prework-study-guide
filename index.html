Prework Study Guide

HTML

HTML (Hypertext Markup Language) is the standard markup language for creating web pages.


Key topics include:

  • Basic Structure
  • Elements and Tags
  • Attributes

CSS

CSS (Cascading Style Sheets) is used for describing the presentation of a document written in HTML.


Key topics include:

  • Selectors
  • Box Model
  • Flexbox and Grid

Git

Git is a version control system for tracking changes in computer files and coordinating work on those files among multiple people.


Key topics include:

  • Repositories
  • Commits
  • Branches

JavaScript

JavaScript is a programming language that enables interactive web pages; it is an essential part of web applications.


Key topics include:

  • Variables and Data Types
  • Functions
  • DOM Manipulation
